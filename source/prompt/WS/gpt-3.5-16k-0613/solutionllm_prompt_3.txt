Question: Sort the following words alphabetically: List: oven costume counterpart
Solution: To sort the list of words alphabetically, we can use the built-in sort() function in Python. This function rearranges the elements of a list in ascending order. 

Here is the solution:

word_list = ['oven', 'costume', 'counterpart']
word_list.sort()

Answer: The answer is costume counterpart oven.
